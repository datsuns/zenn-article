# これはなに？

* 過去のアクションの状況の追跡

| name    | description      |
| ------- | ---------------- |
| Added   | 追加されたばかり |
| Doing   | 実行中           |
| Pending | 保留中           |
| Dropped | 行われなくなった |
| Closed  | 完了             |

* 原典には「Doing」が無いが、きっとこれはガイド作者さんのカスタマイズ
   * 原典的には「Pending」が該当、する？
